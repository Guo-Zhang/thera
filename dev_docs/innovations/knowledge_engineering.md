# 知识工程

## 文化基础

- 系统性思维 vs 碎片化思维
- 显性化偏好 vs 隐性化习惯
- 严谨性追求 vs 差不多就行
- 长期主义 vs 短期功利

## 创新扩散

阶段1：个人工具 → 证明价值
阶段2：小团队试点 → 展示效果  
阶段3：成功案例推广 → 文化扩散
阶段4：制度性支持 → 文化固化

## 原则

反脆弱原则：假设知识图谱是不可靠的，要在后续使用中持续检测，并能够自动或者半自动维护。

## 特性

- 柔性知识模型：既能处理确定事实，也能处理可能性
- 创造性推理引擎：不只是逻辑推导，还要支持叙事合理性判断
- 多版本知识管理：管理不同的情节发展可能性
- 美学约束系统：将"好故事"的原则转化为可检查的约束
