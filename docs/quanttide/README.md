# 基本标准

## 人机协作

这里有一个人机交互的基本原则。大部分系统都只是标注人或者机，但是我们标注了人机协商一致的情况。
绝大多数系统只建模“人 → 机”或“机 → 人”的单向信息流，却忽略了“人 ⇄ 机”之间可以达成的「共同理解」（mutual understanding）——而这恰恰是深度协作的基础。

人类协作的本质不是“传递信息”，而是建立共享心智模型（Shared Mental Model）。

在两人对话中，我们会不断通过“你是指…？”、“对，就是那个！”来校准彼此的理解。
这个过程叫 Grounding（语用学中的“共识建立”）。
💬 Clark & Brennan (1991) 的经典理论：没有 grounding，就没有真正的沟通。

支持渐进式信任：这正是人类专家协作的方式：先对齐基础假设，再高效推进。

它让机器第一次具备了：

承认不确定性的能力（PENDING）
请求确认的谦逊（Negotiation）
尊重共识的纪律（CONFIRMED 后不再质疑）

最好的 AI 不是替你思考，而是和你一起把思考变得清晰。
