# 沟通语境

## 生成语境

输入示例：

```

根据以下对话，提取当前语境，尽可能使用关键词以减少冗余。

类别：小说创作-整理灵感

对话历史：

小公司和小公司之间在大公司的夹缝生存。
我想写的原因是有职场体验想写，但是不是套路化的大厂经验，而是在更广阔的市场博弈。
```

输出示例：

```
类型：职场小说
核心主题：专业精神、冷静博弈
背景设定：小公司 vs 小公司，在大公司夹缝中生存
创作动机：真实职场体验、非套路化（非大厂）
叙事倾向：市场博弈、现实感、去关系户化
语境关键词：中小企生态、竞争策略、职业尊严、灰色地带、务实叙事
```
