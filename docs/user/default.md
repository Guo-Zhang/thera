# 默认活动：增量式被动观察

## 功能介绍

默认活动（增量式被动观察）是一种 AI 辅助写作工具。它能在你写完工作日志后，自动为内容添加结构化批注，帮助你从日志中提取知识，同时保留原文不变。

## 核心特点

- **不改原文**：AI 只添加批注，不修改你的原始内容
- **增量处理**：已处理段落不会重复处理
- **结构化输出**：自动提取标签、提炼、关键要点等

## 使用方法

### 1. 安装

```bash
# 确保已安装 thera
pip install -e .
```

### 2. 配置文件

在项目目录创建 `config.yaml`：

```yaml
opencode_path: /usr/local/bin/opencode
model: o4-mini
max_retries: 3
```

### 3. 运行

```bash
# 对单个文件添加批注
python -m thera.cli /path/to/journal.md
```

### 4. 输出示例

处理前：
```
今天完成了项目规划，确定了三个核心功能。
```

处理后：
```
今天完成了项目规划，确定了三个核心功能。

> 🤖 观察者注
> 🏷️ 标签：#项目进展 #决策
> 💎 提炼：确定三个核心功能，标志项目进入执行阶段
```

## 批注格式说明

| 字段 | 说明 |
|------|------|
| 🏷️ 标签 | 内容分类，如 #核心洞察、#待办、#风险 |
| 💎 提炼 | 标准化知识提取 |
| ⚠️ 状态 | 当前状态，如模糊阶段、进行中 |
| 🔗 关联 | 与其他内容的关联 |
| 🔑 关键 | 关键要点 |
| 🔄 模式 | 模式说明 |

## 注意事项

- 短段落（<20字）不会被处理
- 已添加批注的段落不会重复处理
- 使用 opencode 免费模型，无需 API 密钥
