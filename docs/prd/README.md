# 产品需求文档

## 用户

苹果，深度知识工作者，主业是IT创业者。

## 场景

摆在我面前的课题是长期的，无论是个人还是工作。
当问题很多的时候，我很难在脑子里存所有的信息，就需要忘记很多。下一次再想起来的时候就需要回忆上下文。
我尝试记录，但是随着记录越来越多，我越来越觉得查找和分析也是难题。
主要用于自我探索、工作思考、小说创作三个需要组织大量相互关联的结构化知识的领域。

## 需求

希望这个平台可以帮助我记录我的知识和方法，帮助我减少重复思考和回忆。
通过这个外脑给我节约更多的注意力用来产生想法和吸收想法等更创造性的活动，把加工整合信息的过程交给外脑辅助。

## 功能

分类、检索等知识管理。
自动推理。
澄清输入的模糊知识，减少后期加工成本。
