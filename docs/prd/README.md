# 产品需求文档

## v0.1

概述：TUI实现智能对话摘要。
目标：Message->Note
流程：
1. 人输入消息
2. AI 回复消息。
3. 人反馈 AI 是否可用，分为 yes or no
4. AI 生成便签。yes 合并人机，no 只保留人。


## v0.2

目标：AI增强备忘录+语境摘要
技术栈： Python+Textual

流程：先对话、再落笔
1. 对话
2. 识别对话语境
3. 记录
4. 识别备忘语境

## v1.0

场景：
1. 小说创作
2. 产品研发

需求：
- 最根本的动机：把事情清出大脑，让软件帮我记着。
- 最主要的阻碍：整理加工的成本太高。
- 最想要开发的功能：自动加工整理。

功能：
1. 输入或者导入备忘录。
2. AI 精炼、人工确认、保存。
