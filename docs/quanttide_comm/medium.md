# 沟通媒介`Medium`

Message、Note、Memo（消息、便签、备忘）本质上都是信息的媒介，它们承载内容，但本身不决定意义。
Note 和 Memo的主要区别在于前者只是简单文本，后者有结构。理想生命周期：天级。
Note一般是不需要维护的。不要把精力花在保存火花上，而要花在点燃火堆上。理想生命周期：小时级。
Message 比 Note 更该快速消失。 Message 的理想生命周期：秒级到分钟级
不是“消失”，而是“归档”。归档 = 退出工作区，但保留在系统内。

## Message归档

归档时机：Dialogue 结束 + Note 提取完成后；
归档形式：按会话主题/日期打包，标记为 archived/dialogue/2026-01-02_novel_strategy；
访问方式：需主动搜索或通过 Note/Memo 的反向链接进入；
默认状态：不在主界面、不参与索引、不触发提醒。

## Note归档

归档时机：被 Canvas 处理后（无论升格为 Memo 或废弃）；
归档形式：移入 archive/notes/，保留原始时间戳与上下文引用；
元数据建议：标注结局——→ memo#12 / × discarded / ? pending。
