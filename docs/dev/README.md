# 开发者文档

AI 外脑是一个我自研的类似于 Claude Code 和 Cowork 的 TUI 工具。我设置了默认模式和非默认模式，后者是我设计的专用 System2，比如思考、写作等。default 是 system1.

入口：命令行
交互方式：对话
内部架构：AI 根据需要自动切换到内置模式，比如思考模式。也可以由用户写命令切换。

可选模式：思考模式、写作模式、知识工程模式等等。

默认存储路径：用户根目录下的/thera 文件夹。
存储提交与备份：根据使用习惯再做决定。优先 GitHub，次选阿里云。

备份与安全：禁止直接操作备份平台，这些备份平台手动维护恢复。
充分利用厂商的两阶段验证作为安全墙拦截 AI 的敏感操作。
类似于要更高频的把 AI 当成黑客防着。

## 模块划分及关联

### 认知管理模块

System 1：直接互动并记录
System 2：观察思考活动，评估思考结果等。

### 知识工程模块

从 Thought 提炼 Knowledge 再启发新一轮的思考。

### 认知管理 vs 知识工程

认知管理提供原始素材，知识工程加工素材并提供新的认知问题。
